História dos computadores

Voltar ao topo

O primeiro computador foi criado entre 1943 e 1946 e se chamava Eletronic Numerical Integrator and Computer — ENIAC. No começo, só empresas e certas instituições faziam uso de um computador e, com o passar do tempo, essa tecnologia passou a ser acessada por todos. Por volta de 1980, a IBM liderava o ramo da computação e a Microsoft começava a desenvolver suas ideias e ofereceu o seu próprio sistema operacional para a IBM. A Microsoft acabou vendendo também o seu sistema operacional para outras empresas, garantindo assim a evolução dos computadores para as pessoas. Em 1993, surgiu a World Wide Web, mais conhecida como WWW. Além disso, o celular começava a surgir para atender as necessidades das empresas.

O que são Clients?

Voltar ao topo

Cliente é quem inicia a comunicação com o servidor. O cliente mais conhecido é o navegador. Quando acessamos a internet pelo navegador, uma requisição é enviada pela internet para o servidor no qual esse site está hospedado. O front-end é a parte do site com a qual você pode ver e interagir diretamente para receber os recursos de back-end do sistema. Envolve tudo o que o usuário pode ver, tocar e experimentar. Ou seja, ele envolve tudo que as pessoas que visitam o seu site podem ver e interagir. Esta área envolve linguagens que não são consideradas de programação em si, como o HTML e o CSS, além do JavaScript que é uma linguagem de programação amplamente usada na web e também pode ser vista no back-end.

O que são Servers?

Voltar ao topo

O servidor recebe as requisições e devolve uma resposta ao cliente. O servidor é composto de um ou mais computadores. Nestes computadores podem ser hospedados documentos, imagens, áudio, vídeo, etc. Geralmente o servidor também hospeda programas que atendem as requisições dos clientes e as processam. Chamamos esses programas de web server, ou servidor web. O web server geralmente é capaz de executar aplicações escritas em linguagens como PHP, C#, Java, etc. Portanto, é a parte do site que não faz contato algum com o usuário. Porém, ela é tão essencial quanto.

Como é uma estrutura básica de um HTML?

Voltar ao topo

Um documento HTML tem sua estrutura formada pelas tags < html>, < head> e < body> e a instrução . Esta estrutura informa aos navegadores como eles devem considerar e interpretar estas tags.

Entendendo a diferença entre listas ordenadas e não ordenadas

Voltar ao topo

Listas ordenadas

Uma lista ordenada começa com a tag < ol> e seus respectivos itens da lista ficam dentro da tag < li>. Por padrão as listas ordenadas são ordenadas por números.

Listas desordenadas

Já as Listas desordenadas são iniciadas com a tag < ul> e são representadas por pequenos “bullets”.
